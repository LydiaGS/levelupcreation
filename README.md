# 🚀 LevelUpCreationApp

Application de planification de projet interactive avec post-its, flèches et texte libre.

![React](https://img.shields.io/badge/React-18-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-3-blue)
![Vite](https://img.shields.io/badge/Vite-5-purple)

## ✨ Fonctionnalités

- 📌 **Post-its** - Créez des notes colorées, redimensionnables et éditables
- ➡️ **Flèches** - Reliez vos idées avec des flèches personnalisables
- ✏️ **Texte libre** - Ajoutez du texte avec différentes tailles et couleurs
- 🖱️ **Canvas infini** - Naviguez et zoomez librement
- 🗺️ **MiniMap** - Visualisez l'ensemble de votre projet

## ⌨️ Raccourcis clavier

| Touche | Action |
|--------|--------|
| `V` | Outil sélection |
| `P` | Outil post-it |
| `T` | Outil texte |
| `A` | Outil flèche |
| `H` | Outil déplacement |
| `Suppr` | Supprimer l'élément |
| `Ctrl+D` | Dupliquer |
| `Escape` | Désélectionner |

## 🛠️ Installation locale

```bash
# Cloner le repo
git clone https://github.com/TON_USERNAME/levelupcreationapp.git

# Aller dans le dossier
cd levelupcreationapp

# Installer les dépendances
npm install

# Lancer en développement
npm run dev
```

## 🌐 Déploiement

L'application se déploie automatiquement sur GitHub Pages à chaque push sur `main`.
